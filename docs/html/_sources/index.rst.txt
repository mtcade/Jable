Welcome to Jable
=================================

Jable is a simple too for serializing simple tabular data for casual use on the disk, as an improvement to the .csv format. The JFrame is an unassuming representation of the Json Table (jable). has rows as numbers, columns as strings. There are two reasons to use it over .csv:

    1. It can encode anything serializable to Json as an item in a cell, with support for custom encoders and decoders
